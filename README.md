# OpenClaw Interactive Tutorial

This project is an interactive web app that teaches how OpenClaw works, from first principles through building real workflows.

The product should feel like a guided simulator, not just a prettier copy of the docs.

## Core idea

The app teaches learners how to think in OpenClaw concepts:
- agent
- workspace
- session
- tools
- memory
- channels
- cron
- skills
- subagents
- workflows

It should progressively move the learner from understanding to design.
